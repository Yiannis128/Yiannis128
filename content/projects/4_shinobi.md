This was a remake of the ninja star throwing bonus levels of the Shinobi arcade
where the Shinobi would throw ninja stars at the on-coming ninjas. This project
was built using the Godot game engine.

This project's source code cannot be shared as it contains copyright content
from SEGA. If you are interested in gaining access to the source code (excluding
the art and sounds) then email me.

This project was created for demonstration purposes and is in no way intended to
replace the original game.
