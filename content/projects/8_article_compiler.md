Article compiler is a simple shell script that takes articles written in <a
href="https://en.wikipedia.org/wiki/Markdown">Markdown</a> and
convert them into a HTML file that is suitable to be included in
websites, it also supports an easy template system that will allow
the output to be styled as the rest of the website.

The articles in the [articles](articles.html) page have been
compiled using this script.
