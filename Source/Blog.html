Massive Website Overhaul

As you can see, the website has been updated to feature a much better UX. Before, the website used plain CSS, which was hard to maintain and create largescale changes for.

This new update uses TailWindCSS under the hood for a much more pleasant UX featuring animations, and a much richer style.

Some website elements were re-arranged or removed. The CV page has now been removed (CV can still be requested). Instead, there is a Blog page, which will feature news, and hopefully some useful information in the future!
